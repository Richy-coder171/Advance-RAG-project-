# HR Policy Documents

Place the official Zyro Dynamics HR policy files here before building the index.

Supported formats:
- Markdown: `.md`
- Text: `.txt`
- PDF: `.pdf`
- Word: `.docx`
- CSV: `.csv`
- JSON: `.json`

Do not add fabricated policies for the Kaggle run. The chatbot should answer only from the official policy documents supplied for the assignment.

The `temp_policies/` folder contains temporary HR policy documents for local testing.
